E-Commerce(JAVA) :

- Project involves the development of a dynamic E-commerce website, offering an engaging shopping experience for users. This project is divided into two phases: one tailored for regular users and another dedicated to administrative functions. For regular users, the platform allows seamless browsing and searching for products, facilitating easy additions to the cart. Users can navigate through user-friendly categories and access categorically organized products. On the admin side, a dedicated dashboard empowers administrators to manage product listings, categories, and user accounts. For displaying popup message i have made use of sweatalert.

Technologies Used :

JSP, Servlets, JDBC, Bootstrap, and MySQL.
