Crime Alert System :

The Crime Rate App is a mobile application that provides users with access to crime rate information for specific areas. It caters to individuals moving to new areas and police departments managing crime data. The app features a user-friendly interface and secure data encryption using AES 256. Blockchain technology ensures data integrity and alerts police to any tampering attempts. The app offers real-time crime rate updates, empowering users to make informed decisions about safety. The future scope includes heatmap visualization, crime prevention tips, and advanced analytics. The Crime Rate App aims to enhance public safety and collaboration between users and law enforcement.

Technologies Used :

XML, PHP, JAVA, SQL
